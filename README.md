# ShopHub - E-Commerce Web Application

A React-based e-commerce application built for a school final project.

**Vercel Link:** [ShopHub on Vercel](https://e-commerce-web-application-one.vercel.app)

## Features

- Product browsing with search, filters, and sorting
- Product detail page with reviews and ratings
- Functioning Shopping cart
- User review system with rating updates
- Responsive design for mobile and desktop
- Complete checkout with tax calculation
- Multiple simulated payment methods (Credit Card, PayPal, Bank Transfer)
- Order confirmation page

## Setup

### Running the App

```bash
npm run dev
```

The app runs at `http://localhost:5173`
